    Démarrage Rapide

    Vue d'ensemble

    Ce guide vous permettra d'installer et lancer rapidement le Jeu de la Vie. Pour une installation plus détaillée, consultez le guide d'installation complet.

    Prérequis

    Environnement de développement

    Composant Version minimale Version recommandée
    Compilateur C++ GCC 8+ / MSVC 2019+ GCC 11+ / MSVC 2022+
    CMake 3.10+ 3.20+
    SFML 2.5+ 2.6+
    Git 2.0+ 2.34+

    Espace disque

    • Sources : ~10 MB
    • Build : ~100 MB
    • Installation : ~50 MB

    Installation

    Linux (Ubuntu/Debian)

    # Mise à jour du système
sudo apt update

# Installation des dépendances
sudo apt install -y \
    build-essential \
    cmake \
    libsfml-dev \
    git

# Clonage et compilation
git clone https://github.com/votre-repo/jeu-de-vie
cd jeu-de-vie
mkdir build && cd build
cmake ..
make -j$(nproc)

    Windows

    1. Installer les outils
    2. Visual Studio avec C++
    3. CMake
    4. SFML 2.5+

    5. Git

    6. Compiler 

      # Cloner le projet
git clone https://github.com/votre-repo/jeu-de-vie
cd jeu-de-vie

# Créer et configurer le build
mkdir build
cd build
cmake ..

# Ouvrir et compiler avec Visual Studio
start JeuDeLaVie.sln

    macOS

    # Installer les dépendances avec Homebrew
brew update
brew install cmake sfml git

# Cloner et compiler
git clone https://github.com/votre-repo/jeu-de-vie
cd jeu-de-vie
mkdir build && cd build
cmake ..
make -j$(sysctl -n hw.ncpu)

    Lancement rapide

    Mode graphique (défaut)

    # Lancer avec paramètres par défaut
./jeu-de-vie

# Spécifier la taille de la grille
./jeu-de-vie --width 100 --height 100

    Mode console

    # Mode console simple
./jeu-de-vie -c

# Mode console avec options
./jeu-de-vie -c --speed 2 --toroidal

    Commandes essentielles

    Contrôles généraux

    Touche Action Description
    Espace Play/Pause Démarre/Arrête la simulation
    R Reset Réinitialise la grille
    Q Quitter Ferme l'application
    +/- Vitesse Ajuste la vitesse de simulation

    Interactions souris (Mode graphique)

    Action Effet
    Clic gauche Active/Désactive une cellule
    Clic droit Place un motif prédéfini
    Molette Zoom in/out
    Glisser Déplace la vue
