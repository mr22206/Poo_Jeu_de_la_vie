    Changelog¶

    Toutes les modifications notables du projet seront documentées dans ce fichier.

    Le format est basé sur Keep a Changelog, et ce projet adhère au Semantic Versioning.

    [1.2.0] - 2024-03-15¶

    Ajouté¶

    • Mode debug avec visualisation des états internes
    • Nouveaux motifs prédéfinis (canon à planeur, pulsars)
    • Support pour les fichiers de configuration YAML
    • Interface en ligne de commande améliorée

    Modifié¶

    • Optimisation majeure du moteur de simulation
    • Refonte de l'interface utilisateur graphique
    • Amélioration de la gestion de la mémoire

    Corrigé¶

    • Correction des fuites mémoire dans le gestionnaire de motifs
    • Résolution du bug de synchronisation en mode multi-thread
    • Correction de l'affichage des statistiques en temps réel

    [1.1.0] - 2024-02-01¶

    Ajouté¶

    • Mode console avec support couleur
    • Système de sauvegarde/chargement de parties
    • Statistiques en temps réel
    • Documentation utilisateur complète

    Modifié¶

    • Amélioration des performances de rendu
    • Mise à jour de la bibliothèque SFML
    • Simplification de l'API publique

    Déprécié¶

    • Ancien format de fichier de sauvegarde (.sav)
    • Mode de rendu OpenGL legacy

    Corrigé¶

    • Bug de bordure infinie
    • Problème de performance avec les grandes grilles
    • Crash lors du redimensionnement de la fenêtre

    [1.0.0] - 2024-01-01¶

    Ajouté¶

    • Première version stable
    • Interface graphique basique
    • Motifs de base (planeur, bloc, clignotant)
    • Contrôles souris et clavier
    • Système de grille infinie
    • Mode pause/reprise
    • Contrôle de la vitesse de simulation

    Notes de version¶

    • Version initiale du Jeu de la Vie
    • Support pour Windows, Linux et macOS
    • Documentation de base

    [0.9.0] - 2023-12-15¶

    Ajouté¶

    • Version beta publique
    • Tests unitaires et d'intégration
    • Pipeline CI/CD
    • Documentation préliminaire

    Connu¶

    • Problèmes de performance sur grandes grilles
    • Interface utilisateur basique
    • Support limité des motifs
