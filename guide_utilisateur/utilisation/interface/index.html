    Interface Utilisateur

    Introduction

    Le Jeu de la Vie propose deux interfaces distinctes : un mode graphique moderne avec SFML et un mode console traditionnel. Chaque mode offre ses avantages spécifiques selon vos besoins.

    Modes d'affichage

    Mode Graphique (SFML)

    • Grille interactive en 2D
    • Cellules colorées :
    • Vivantes : blanc
    • Mortes : noir
    • Sélectionnées : bleu
    • Interface de contrôle intuitive
    • Affichage des statistiques en temps réel

    Mode Console (ASCII)

    • Représentation en caractères ASCII
    • Symboles :
    • Cellule vivante : '*'
    • Cellule morte : ' '
    • Bordure : '+' et '-'
    • Interface minimaliste mais fonctionnelle

    Contrôles

    Commandes générales

    Touche Action Description
    Espace Play/Pause Démarre/Arrête la simulation
    R Reset Réinitialise la grille
    Q Quitter Ferme l'application
    S Sauvegarder Enregistre l'état actuel
    L Charger Charge une configuration

    Contrôles de simulation

    Touche Action Description
    + Augmenter vitesse Accélère la simulation
    - Diminuer vitesse Ralentit la simulation
    T Mode torique Active/Désactive les bords infinis

    Interactions souris (Mode graphique)

    Action Effet Description
    Clic gauche Activer/Désactiver Modifie l'état d'une cellule
    Clic droit Placer motif Insère un motif prédéfini
    Molette Zoom Ajuste l'échelle de la grille
    Drag Déplacer Déplace la vue (avec zoom)

    Fonctionnalités

    Sauvegarde/Chargement

    • Format : fichier texte simple
    • Contenu :
    • Dimensions de la grille
    • États des cellules
    • Configuration actuelle
    • Compatibilité entre modes

    Options de simulation

    • Vitesse : 1-10x
    • Mode torique : grille infinie
    • Statistiques temps réel :
    • Population vivante
    • Numéro de génération
    • Images par seconde

    Dépannage

    Problèmes courants

    • Fenêtre ne s'ouvre pas : Vérifier installation SFML
    • Contrôles ne répondent pas : Vérifier focus fenêtre
    • Performances faibles : Réduire taille grille/vitesse

    Raccourcis clavier

    • Ctrl+S : Sauvegarde rapide
    • Ctrl+L : Dernier fichier chargé
    • Ctrl+R : Reset avec confirmation
