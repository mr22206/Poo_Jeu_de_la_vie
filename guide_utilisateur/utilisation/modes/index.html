    Modes de Jeu

    Introduction

    Le Jeu de la Vie propose deux modes d'exécution distincts, chacun adapté à des besoins spécifiques. Cette section vous aide à choisir et utiliser le mode le plus approprié.

    Comparaison des modes

    Caractéristique Mode Graphique Mode Console
    Interface GUI moderne ASCII simple
    Performance Moyenne Excellente
    Mémoire ~50-100MB ~5-10MB
    Interaction Souris + Clavier Clavier seul
    Usage idéal Exploration visuelle Tests rapides

    Mode Graphique (SFML)

    Caractéristiques

    • Rendu fluide 60 FPS
    • Zoom dynamique (x0.5 - x5)
    • Grille redimensionnable
    • Statistiques en temps réel
    • Population active
    • Génération actuelle
    • Performance (FPS)

    Avantages

    ✅ Léger et rapide ✅ Fonctionne partout ✅ Idéal en SSH ✅ Performances optimales

    Limitations

    ❌ Visibilité réduite ❌ Contrôles limités ❌ Grille plus petite

    Guide de choix

    Choisir le mode graphique si :

    • Vous découvrez le jeu
    • Vous souhaitez expérimenter visuellement
    • Vous avez besoin de contrôles précis
    • Les ressources système ne sont pas limitées

    Choisir le mode console si :

    • Vous êtes en SSH
    • Vous faites des tests rapides
    • Vous avez des ressources limitées
    • Vous préférez la simplicité
