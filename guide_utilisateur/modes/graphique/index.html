    Mode Graphique¶

    Présentation¶

    Le mode graphique offre une interface visuelle interactive permettant de : - Visualiser l'évolution en temps réel - Interagir directement avec la grille - Observer les motifs émergents - Modifier les paramètres dynamiquement

    Lancement¶

    ./jeu-de-vie -g <fichier_config> [options]

    Options graphiques¶

    • -w <largeur> : Largeur de la fenêtre (défaut: 800)
    • -h <hauteur> : Hauteur de la fenêtre (défaut: 600)
    • -s <taille> : Taille des cellules (défaut: 10px)
    • -f <fps> : Limite de FPS (défaut: 60)

    Interface utilisateur¶

    Éléments d'interface¶

    • Grille principale
    • Barre d'outils
    • Panneau de statistiques
    • Menu des motifs
    • Contrôles de simulation

    Contrôles¶

    Action Commande
    Clic gauche Activer/désactiver cellule
    Clic droit Menu contextuel
    Molette Zoom in/out
    Espace Pause/Reprise
    R Réinitialiser
    S Sauvegarder
    L Charger motif
    +/- Vitesse
    Échap Quitter

    Fonctionnalités avancées¶

    Gestion des motifs¶

    1. Appuyez sur L pour entrer en mode insertion
    2. Sélectionnez un motif (1-9)
    3. Positionnez avec la souris
    4. Clic gauche pour placer

    Motifs disponibles¶

    • 1: Glider
    • 2: Blinker
    • 3: Block
    • 4: Beehive
    • 5: Pulsar
