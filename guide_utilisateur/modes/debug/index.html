    Mode Debug¶

    Présentation¶

    Le mode debug est conçu pour : - Analyser le comportement du jeu - Détecter les anomalies - Optimiser les performances - Valider les règles d'évolution

    Activation¶

    ./jeu-de-vie -d <fichier_config> [options]

    Options de debug¶

    • --step : Exécution pas à pas
    • --stats : Affichage statistiques
    • --trace : Log détaillé
    • --profile : Profilage
    • --check : Validation des règles

    Fonctionnalités¶

    Analyse pas à pas¶

    [DEBUG] Génération 42, Cellule (3,4)
- Voisins vivants: 3
- État actuel: mort
- Prochain état: vivant
- Règle appliquée: naissance

    Statistiques en temps réel¶

    • Temps de calcul par génération
    • Utilisation mémoire
    • Nombre de cellules actives
    • Points chauds d'activité

    Profilage¶

    # Activation du profilage
./jeu-de-vie -d config.txt --profile

# Résultats
Performance Report:
- Update: 0.5ms
- Render: 1.2ms
- Memory: 24MB

    Outils de débogage¶

    Console de debug¶

    • Commandes disponibles
    • Inspection des états
    • Modification en temps réel
    • Breakpoints

    Logs¶

    Niveaux de log : - ERROR : Erreurs critiques - WARN : Avertissements - INFO : Informations générales - DEBUG : Détails techniques - TRACE : Données exhaustives
