    Guide de dépannage¶

    Problèmes d'installation¶

    Erreurs de compilation¶

    CMake non trouvé¶

    # Erreur
CMake not found

# Solution
sudo apt install cmake  # Ubuntu/Debian
brew install cmake     # macOS

    SFML manquant¶

    # Erreur
Could not find SFML

# Solution
sudo apt install libsfml-dev  # Ubuntu/Debian
brew install sfml            # macOS

    Erreurs de linkage¶

    Bibliothèques manquantes¶

    # Erreur
error while loading shared libraries

# Solution
export LD_LIBRARY_PATH=/usr/local/lib:$LD_LIBRARY_PATH
sudo ldconfig

    # Vérifier les permissions
ls -l ~/.local/share/jeu-de-vie/

# Corriger les permissions

    Outils de diagnostic¶

    Logs système¶

    # Afficher les logs
tail -f ~/.local/share/jeu-de-vie/logs/error.log

# Filtrer les erreurs
grep "ERROR" ~/.local/share/jeu-de-vie/logs/error.log

    Tests de diagnostic¶

    # Test système
./jeu-de-vie --diagnose

# Test graphique
./jeu-de-vie --test-renderer

# Test performance
./jeu-de-vie --benchmark
